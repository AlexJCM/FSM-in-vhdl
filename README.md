
## **Diseño de un controlador de máquina expendedora**

<p align="center"> 
<img src="https://i.postimg.cc/Sxn9GcdK/vmachine.png" alt="drawing"/>
</p>

La máquina expendedora tiene las siguientes características (descripción textual): 

La máquina expendedora vende solo un tipo de producto, y cada producto cuesta $ 0.25;
- Sólo se aceptan $ 0.25 monedas;
- Una moneda válida de $ 0.25 pesa 5.670 g;
- Las monedas no válidas se rechazan automáticamente, y el circuito de control no necesita hacer nada al respecto;
- Hay una ranura para insertar $ 0.25 monedas en la máquina expendedora;
- Hay un circuito de detección de monedas que emite "1" cada vez que se inserta una moneda de $ 0.25 en la ranura, o "0" de lo contrario
- Hay una balanza digital que genera "1" cuando se detecta un peso de 5.670 g (o más);
- Hay un ‘botón de entrega’ ’que emite‘ 1 ’cada vez que se presiona;
- Hay un ‘botón de retorno de moneda’ ’que emite‘ 1 ’cada vez que se presiona;
- Hay una ‘taza de devolución de monedas’ para recoger las monedas expulsadas por el ‘botón de devolución de monedas’ (y también las monedas rechazadas);
- Hay una puerta donde el usuario puede recoger su producto, después de haber presionado el "botón de entrega";
- Cuando la máquina expendedora se queda sin productos, se cierra y el circuito de control no necesita hacer nada al respecto.
